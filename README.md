## 📁 Repository Structure

```
src/
├── Basics/           # Language fundamentals, Maths, Hashing, Recursion basics
├── Sorting/          # Bubble, Selection, Insertion, Merge, Quick Sort
├── Arrays/           # Easy → Medium → Hard array problems
├── BinarySearch/     # 1D, 2D arrays, Search Space problems
├── Strings/          # Basic and Medium string problems
├── LinkedList/       # Single LL, Double LL, Medium & Hard problems
├── Recursion/        # Pattern-wise recursion problems
├── BitManipulation/  # Bitwise concepts and problems
├── StackAndQueues/   # Learning, Pre/In/Post-fix, Monotonic Stack
├── SlidingWindow/    # Sliding Window & Two Pointer problems
├── Heaps/            # Learning, Medium, Hard heap problems
├── Greedy/           # Greedy algorithms
├── BinaryTrees/      # Binary tree traversals and problems
├── BST/              # Binary Search Tree problems
├── Graphs/           # BFS, DFS, shortest paths, topological sort
└── DP/               # Dynamic Programming (1D, 2D, DP on strings/trees)
```
